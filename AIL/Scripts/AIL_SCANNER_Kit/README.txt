
=====================================
AIL SCANNER AVANÇADO - ToCA Race Driver 3
=====================================

Este scanner tenta:
- Ler coordenadas (XYZ) de nós
- Detectar blocos de conexões (ligações entre nós)
- Exportar tudo para CSV
- Gerar meta.json com infos úteis

===========================
USO
===========================
1) Coloque seu arquivo 'input.ail' na mesma pasta.
2) Clique em 'run_scanner.bat'.
3) Abra os arquivos 'vertices.csv', 'connections.csv', 'meta.json'.

===========================
NOTA
===========================
Este scanner é genérico. Pode precisar de ajustes 
para blocos específicos de cada pista.

===========================
AVISO
===========================
Sempre teste o output antes de reconstruir um AIL.
